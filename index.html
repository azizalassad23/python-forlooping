Di Python, kita menggunakan mantra for untuk menyuruh komputer melakukan hal membosankan itu secara otomatis.

üìú Scroll of Knowledge

Ini adalah struktur mantra dasar untuk melakukan pengulangan:

for x in range(5):
    print("Serangan Laser!")

Apa itu range(5)?
Itu adalah generator angka. Dia akan menghitung: 0, 1, 2, 3, 4.
PENTING Komputer selalu mulai menghitung dari 0, bukan 1!

Taktik 1: Power Up Bertingkat

Ingin menambah kekuatan (XP) setiap kali naik level?

total_xp = 0
for level in range(1, 6):
    print(f"Naik ke level {level}")
    total_xp = total_xp + 100

print(f"Total XP: {total_xp}")

Taktik 2: Inventory Check

Mengecek isi tas (List) item satu per satu:

inventory = ["Pedang", "Potion", "Peta"]

for item in inventory:
    print(f"Kamu menggunakan {item}!")
